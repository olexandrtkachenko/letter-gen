# X-Stack Letter Generator

Professional letter generator for X-Stack Project Education Course based on Excel data.

## Features

- 9 different letter templates
- Automatic data parsing from Excel
- Real-time preview
- Copy HTML or download as file
- Outlook-compatible HTML generation

## Templates

1. **Initial Email** - Initial email for a new Run
2. **Status Letter** - Status letter with detailed student information
3. **GitHub Copilot** - GitHub Copilot license management
4. **Status Update** - Status update with automatic counting
5. **AWS Accounts Creation Request** - Request for AWS account creation
6. **KubeRocketCI Accounts Creation Request** - Request for KubeRocketCI account creation
7. **Final Letter** - Final letter with summary and statistics
8. **Run Completion** - Final status for Run completion
9. **Instructions** - Complete usage guide

## Usage

1. Open `index.html` in your browser
2. Select the desired template
3. Fill in the form fields
4. Paste Excel data if required
5. Preview the generated letter
6. Copy HTML or download the file
7. Use the Instructions page for detailed guide
